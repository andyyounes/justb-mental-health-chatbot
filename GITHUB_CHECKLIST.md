## ⚠️ Common Issues

### Issue 1: Files Not Showing on GitHub

**Cause**: `.gitignore` may be excluding important files

**Solution**: Check `.gitignore` and ensure no source files are accidentally ignored

### Issue 2: Large File Warning

**Cause**: File size > 50MB (GitHub limit is 100MB)

**Solution**: 
- Use Git LFS for large files
- Or move large files to Supabase Storage
- JustB should not have this issue (all files are small)

### Issue 3: Sensitive Data Exposed

**Cause**: Accidentally committed `.env` or secrets

**Solution**:
1. Remove from repository:
   ```bash
   git rm --cached .env
   git commit -m "Remove sensitive files"
   git push
   ```
2. Rotate all exposed secrets immediately!
3. Update `.gitignore`

### Issue 4: Merge Conflicts

**Cause**: Multiple people or multiple machines editing

**Solution**:
```bash
git pull --rebase
# Resolve conflicts
git add .
git rebase --continue
git push
```
